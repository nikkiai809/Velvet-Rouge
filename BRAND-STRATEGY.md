# VELVET ROUGE — Brand Experience Strategy

---

## 1. Emotional Strategy

**Core truth:** There are people who stay awake because they feel it's not over yet. Something is about to happen. They don't know what. But they can feel it.

Velvet Rouge exists for those people.

Not as a brand. As a recognition.

The emotional arc follows a single night — from the restlessness of dusk to the quiet transformation of dawn.

**Arc:**
- **Dusk (22:00):** Restlessness. The night is a blank page.
- **Late Night (00:00):** Longing. The world is asleep. You are not.
- **Deep Night (02:00):** Obsession. Thoughts become dangerous.
- **The Witching Hour (03:30):** Fantasy. Reality bends.
- **Pre-Dawn (05:00):** Possibility. The shift.
- **Dawn (06:00):** Quiet transformation. Not a resolution. A beginning.

**Emotional pillars:**
- *Anticipation:* The moment before
- *Longing:* The ache for what could be
- *Obsession:* The thought that won't leave
- *Desire:* The fire beneath the stillness
- *Wonder:* The intoxication of "what if"

---

## 2. Positioning

**In a world that sells solutions, Velvet Rouge sells questions.**

It does not promise answers. It promises the electric space between hope and uncertainty.

**Brand positioning statement:**

*For those who feel most alive when the world is asleep, Velvet Rouge is the recognition that your restlessness is not emptiness — it is the beginning of something.*

**Competitive landscape (anti-positioning):**
- We are not a lifestyle brand. We are a feeling.
- We are not a luxury good. We are a lens.
- We are not an experience. We are a night.

**Brand territory:**
- Emotional luxury
- Poetic technology
- Cinematic intimacy

## 5. Wireframe per Section

### Opening Sequence
- Full viewport, pure black
- Centered text: white, serif, size 3rem→5rem
- Timing: 3s black → 2s fade in "It's late." → 1.5s hold → 2s fade "Good."
- Total: ~8-9 seconds
- Skip mechanism: click/tap anywhere exits sequence
- After sequence: gentle fade to hero

### Hero
- Full viewport, dark gradients
- Centered title: "Velvet" / "Rouge"
- Rouge in gradient red-gold
- Subtitle: ultra-light tracking
- Time indicator: top-right corner
- Scroll down arrow: bottom center

### Night Section
- Split layout: text left, poetry right
- Subtle parallax
- Manifesto fragment 1: "The nighttime brings..."
- Decorative vertical divider

### Longing Section
- Full-screen, centered
- Rotating words: Longing / Obsession / Desire
- Each word fades with poetic subtext
- Interactive prompt at bottom

### What If Engine
- Dark, intimate
- Centered question: "What are you longing for?"
- Text input styled as elegant form
- After submit: "What if it happens?" with their answer
- Cinematic transition between states

### Tension Section
- Split screen: Reality | Fantasy
- Animated balance scale
- Pairs rotate automatically
- User can click to change pairs

### Letters Never Sent
- Vertical scrollable list
- Each letter fades in on scroll
- Minimal, poetic, emotional
- Background: darker than dark

### Archetypes
- Grid of 6 cards
- Each has: title, description, icon (geometric)
- Hover reveals deeper text
- Collectively: "You are not alone."

### Dawn Transition
- Gradient shifts from night-950 to warm amber
- Introduces gold light effects
- Text minimal: "Perhaps tonight..."

### Invitation
- Centered, warm
- Email form: minimal, elegant
- Success state: poetic response

## 7. Typography

**Primary: Playfair Display**
- Weights: 300 Light, 400 Regular, 500 Medium, 600 SemiBold, 400 Italic
- Usage: Headlines, hero text, emotional copy
- Feeling: Editorial, European, timeless

**Secondary: Inter**
- Weights: 200 ExtraLight, 300 Light, 400 Regular, 500 Medium
- Usage: Body text, captions, UI elements, form inputs
- Feeling: Clean contrast, modern restraint

**Size Scale:**
- Hero title: clamp(3rem, 10vw, 10rem)
- Section h2: clamp(2rem, 5vw, 5rem)
- Body: clamp(0.875rem, 1.2vw, 1rem)
- Captions: 0.75rem
- Micro (tracking): 0.625rem

**Letter-spacing:**
- Headlines: 0.05em to 0.15em
- Uppercase: 0.3em to 0.4em
- Body: 0.02em

## 8. Animation System

**Philosophy:** Slow, organic, inevitable. Nothing should feel triggered. Everything should feel like it was always going to happen.

**Timing curve:** `cubic-bezier(0.25, 0.1, 0.25, 1)` — The "elegance curve"

**Animation types:**

| Type | Duration | Delay | Element |
|------|----------|-------|---------|
| Fade in | 2s | 0.5-3s | Opening text |
| Text reveal | 1.5s | 0.3s per word | Manifesto fragments |
| Scale in | 1.8s | 0.5s | Hero title |
| Slide up | 1.5s | 0.2s | Section content |
| Rotating word | 0.8s | — | Longing animation |
| Progress bar | 0.7s | — | Section indicator |
| Light shift | 4s | — | Dawn transition |
| Grain overlay | infinite | — | Texture |

**Microinteractions:**
- Custom cursor: 8px dot + 40px ring (desktop only)
- Cursor ring expands on hover over interactive elements
- Text links: underline grows from center on hover
- Form input: border glow on focus
- Archetype cards: subtle lift on hover (translateY -4px)
- Letters: each letter fades in with staggered delay as user scrolls
- What If: gentle pulse on the input field when empty
- Dawn: light gradually warms over 3 section heights

## 9. Immersive Features Detail

### Opening Sequence
A critical emotional anchor. The user arrives and is met with silence and darkness. No logo. No navigation. No "skip intro" button aggressively placed. Just a moment.

The text should feel typed by thought, not by machine.

After completion, the sequence dissolves into the hero section with a subtle light bloom.

### Living Time System
A small display in the top-right corner showing:
```
02:47
The hour when thoughts become dangerous.
```

Time-based messages:
- 18:00-21:00: "The evening is preparing something."
- 21:00-23:00: "The night is still young."
- 23:00-01:00: "This is when dreamers wake up."
- 01:00-03:00: "Most people are asleep. You're not."
- 03:00-05:00: "This is where ideas become dangerous."
- 05:00-07:00: "The world is about to change."
- 07:00-18:00: "The night is waiting."

### What If Engine
Two states:
1. **Prompt:** "What are you longing for?" with an elegant textarea
2. **Response:** User's text is echoed back: "What if [user's answer] happens?"

The transition between states uses a slow crossfade. The response should feel like a revelation, not a confirmation dialog.

Store the response in localStorage so returning users see their own "what if" instead of the prompt.

### Letters Never Sent
Visual approach: each letter is a single line, centered, with a small "— [year]" or "— [city]" attribution.

The letters should auto-advance every 4 seconds BUT respond to scroll — if user scrolls, the advance pauses for 8 seconds.

### Archetypes
Six cards in a 3×2 grid (2×3 on mobile).

Each card:
```
┌──────────────────┐
│  ○ [geometric]   │
│                   │
│  THE DREAMER      │
│                   │
│  Those who build  │
│  worlds before    │
│  morning comes.   │
│                   │
│  [hover reveals]  │
│  "I close my eyes │
│   and I'm already │
│   somewhere else."│
└──────────────────┘
```

## 10. Technical Architecture

### Stack
- Next.js 14 (App Router)
- TypeScript (strict mode)
- TailwindCSS (custom theme)
- Framer Motion (animations, scroll, gesture)
- No UI library — everything is custom

### Performance targets
- Lighthouse: 90+ Performance, 95+ Accessibility
- First Contentful Paint: <1.5s
- Time to Interactive: <2.5s
- Bundle size: <150KB JS

### Component tree
```
RootLayout
├── GrainOverlay
├── Cursor
├── TimeIndicator (persistent)
├── SectionProgress (persistent)
├── OpeningSequence → Hero
│   ├── SequenceFrame
│   └── SkipButton
├── SectionNight
├── SectionLonging
├── WhatIfEngine
├── SectionTension
├── SectionLetters
├── SectionArchetypes
│   └── ArchetypeCard × 6
├── SectionDawn
├── SectionExperience
└── Footer
```

### Responsive strategy
- Mobile-first CSS
- Custom cursor only on desktop (lg:)
- Grid layouts collapse to single column
- Typography scales with clamp()
- Touch interactions replace hover microinteractions

### SEO
- Proper meta tags, OG tags
- Semantic HTML structure
- No client-side rendering for critical text
- Sitemap-ready
